# Always a meaningful task

This goal is that all individuals always have a task to do that is meaningful and shapes the world in the way that they think it is right. This goal becomes relevant in a hypothetical world where every problem is solved by a [superintelligence](../../../topics/ai/singularity/superintelligence.md). When all goals are reached live might become meaningless so this goal is that this state never occurs and there is always something to do.
