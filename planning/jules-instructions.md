# Jules instructions

This file contains instructions for [google-jules](../topics/ai/agents/google-jules.md).

## Instruction for Jules

Please take a look at the file [jules-tasks.md](jules-tasks.md). This file contains a list of liks to files with tasks. For each link open the according markdown file and execute the task that is described in the file. After the task was completed successfully or after it could not be completed change the status at the bottom of the tasks markdown file accordingly. Then remove the link to the task file from [jules-tasks.md](jules-tasks.md). and move this link ether to [completed-tasks.md](completed-tasks.md) or to [failed-tasks.md](failed-tasks.md).
