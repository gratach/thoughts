# Customizable poster website
A website that lets you create multiple customizable posters by adjusting the parameters of the image generation and including content that should be used for the generation process.

It could include multiple different types of customizble posters like:
- The spiral calendar of the days of a life as already implemented in the [spiralife project](https://github.com/gratach/spiralife) ([website](https://spiralife.debablo.de/))
- [Escher style thoughts people world zoom calendar](escher-style-thoughts-peope-wotld-zoom-calendar.md)
- [Generated image that includes code that generated it](generated-image-that-includes-code-that-generated-it.md)
- [Git repo poster](git-repo-poster.md)
- [Planet constellation calendar](planet-constellation-calendar.md)

The website could be open source but include some function to buy a printed version of the generated calendar and ship it to the users address. It cold include multiple price options like `"production costs only price"`, `"fair price"` and `"support development price"`.
