# vim

Vim is a text editor for console.

## Navigation:

h, j, k, l: move cursor
g + {j, k}: move up and down wrapped lines
i, a : start writing
esc : stop writing
:wq : save and exit
:!q : exit without save
v : start highlighting
y : copy
c : cut
p : paste

## Copy and paste from and to system clipboard

To activate pasting Type
`ESC :set paste`
You can then paste with `Ctrl Shift V`

To select a Text for copying press `Shift` and select with mouse.
You can then copy with `Ctrl Shift C`
