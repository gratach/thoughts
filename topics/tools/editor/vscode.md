# VS Code

VS Code is a coding IDE from Microsoft.

[Official website](https://code.visualstudio.com/)

VS Code is based on a open source project and includes telemetry. A version of the same software without telemetry and tracking is distributed as [Codium](codium.md)

## Activate automatic file saving

As described [here](https://dev.to/pexlkeys/how-to-enable-auto-save-in-visual-studio-code-5fj4):
Go to File > Preferences > Settings
In the search bar enter `auto save`
Select `afterDelay`
