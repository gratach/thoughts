# Dunin DB

A dunin db is an entity that stores the [dunin-nodes](../../graph/organizing/dunin-node.md) that an [dunin-agent](../../code/architecture/dunin-agent.md) is aware of.
It keeps track of all the all the known nodes with their [dunin-ids](../adresses/dunin-id.md).
