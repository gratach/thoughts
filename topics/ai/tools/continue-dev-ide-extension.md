# Continue dev ide extension

Continue ([official website](https://www.continue.dev/)) offers a coding copilot that can do code completion and make changes in your codebase based on your chat input.

## Installation

It can be installed as an extension to [Codium](../../tools/editor/codium.md) 
```
codium --install-extension continue.continue
```
## Telemetry

I was not able to disable the telemetry despite of the [online instructions](https://docs.continue.dev/customize/telemetry)
