# Redirect spam mails

Spam and phishing mails can be redirected to one of these mail addresses:
The mail header can be included.

[Police Niedersachsen](https://www.polizei-praevention.de/themen-und-tipps/serviceangebot/spammails-zur-polizei): `trojaner[a_t]polizeilabor.de`

[Bayernportal](https://www.bayernportal.de/dokumente/leistung/1337129445200?localize=false):  `phishing[a_t]verbraucherzentrale.nrw`
